# 🛡️ SKaddTrap — v1.0 (VioTrap Addon)
**Підтримувана версія гри:** `1.16.5+`  
**Основний плагін:** `VioTrap`

---

### 📖 Про аддон
**SKaddTrap** — це спеціалізоване розширення для плагіна **VioTrap**. Він створений для адміністраторів, які хочуть додати рівень контролю та балансу на своїх Анархія-серверах. 

Аддон виправляє популярний абуз, коли гравці ставлять автоматичні пастки безпосередньо на івентах, біля містичних скринь або спавнерів, повністю блокуючи доступ до них. **SKaddTrap** інтегрується в процес використання предметів і забороняє активацію пасток у радіусі важливих об'єктів.

---

### ✨ Ключові особливості

* **🤝 Повна сумісність з VioTrap**
  Використовує унікальний алгоритм "тимчасового вилучення предмета". Коли гравець намагається активувати пастку в забороненій зоні, аддон на 1 тік приховує предмет від `VioTrap`, запобігаючи його спрацьовуванню, а потім повертає назад у руку гравця.
* **🚫 Блокування в радіусі (Anti-Abuse)**
  Запобігає встановленню "пластів" та "трапок" біля стратегічних блоків. Це змушує гравців битися чесно, а не просто закидувати ворога коробками біля скринь з лутом.
* **⚡ Оптимізована логіка**
  Аддон працює на найвищому рівні пріоритету подій, що гарантує 100% скасування дії ще до того, як основний плагін на пастки встигне їх створити.
* **🔧 Гнучкий список блоків**
  Через конфіг можна заборонити ставити пастки біля:
    * Маяків (Beacons)
    * Містичних скринь
    * Спавнерів мобів
    * Столів чарування

---

### ⚙️ Налаштування (config.yml)

```yaml
# Радіус перевірки навколо гравця (в блоках)
radius: 5

# Список блоків-тригерів, які забороняють ставити пастку поруч
restricted-blocks:
  - "CHEST"
  - "TRAPPED_CHEST"
  - "BEACON"
  - "SPAWNER"
  - "ENCHANTING_TABLE"

# Повідомлення, яке побачить гравець при скасуванні
message: "&c[!] Тут надто мало місця або поруч важливий об'єкт для використання пастки!"
